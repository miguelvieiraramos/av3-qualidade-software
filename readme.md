# Avaliação 3 - Princípios de Qualidade de Software 

# Setup do ambiente

## Criar um ambiente virtual
``` 
python -m venv .venv
```

## Entrar no ambiente virtual
```
source .venv/bin/activate
```

## Instalar packages
```
pip install -r requirements.txt
```

## Rodar os testes
```
pytest
```
